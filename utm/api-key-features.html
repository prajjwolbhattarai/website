AI-Powered Spelling Suggestions

By providing your own Google AI API key, you enable a powerful feature that helps you maintain pristine data quality in your analytics. The tool will use the Gemini model to intelligently check your UTM parameters for common spelling mistakes.

Why is this important?

A simple typo can fragment your campaign data. For example, if some links use `utm_source=facebook` and others use `utm_source=facbook`, analytics platforms will treat them as two separate sources. This makes it difficult to get an accurate, consolidated view of your campaign's performance. The AI assistant helps prevent these costly mistakes before they happen.

How to Get and Use Your API Key

  1. Get a free API Key: Visit Google AI Studio to generate your own API key.
  2. Enable the feature: In the UTM Builder application, there is a field to enter your API key. Simply paste your key there.
  3. Start building: The tool will now automatically check your parameters as you type. If a likely error is found, a suggestion will appear, which you can accept or ignore.
