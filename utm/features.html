UTM Builder & Validator Features

A powerful, portable, and offline-first toolkit for marketers.

Core Tools

  • UTM Builder: Quickly generate single, error-free UTM links with all standard and optional parameters.
  • Bulk Generator: Create hundreds of link variations at once by providing lists of URLs and parameters. Perfect for large-scale campaigns and A/B testing.
  • Link Validator: Paste a list of existing URLs to instantly check them for missing or empty required UTM tags (`utm_source`, `utm_medium`, `utm_campaign`).

Workflow & Data Management

  • Generated Link List: All created links appear in a clean, organized list, ready for use or export.
  • Export to CSV: Download your entire link list as a CSV file with a single click, ready to be archived or shared.
  • Google Sheets Integration: Automatically save all generated links to a Google Sheet in your own Google Drive for a persistent, collaborative log.
  • Parameter Consistency Checker: When connected to Google Sheets, the tool checks if a parameter you're using already exists, helping you avoid typos and keep your analytics data clean.

Portability & Accessibility

  • Fully Offline: The core application is a single HTML file. Save it to your computer and use it anywhere, with or without an internet connection.
  • No Installation Needed: Runs directly in any modern web browser. No complex setup or dependencies required.
  • Responsive Design: The interface is designed to work beautifully on desktops, tablets, and mobile devices.

Optional AI Enhancements

  • AI-Powered Spelling Suggestions: By adding a free Google AI API key, you can enable intelligent suggestions to catch common typos in your campaign parameters, ensuring maximum data accuracy.
